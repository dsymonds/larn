Larn 12.5

A Go port of Larn, by David Symonds.

What is it?

Larn is a roguelike game written by Noah Morgan in 1986. This is a web page for my work to restore the game and get it running on modern computers.

A roguelike is a genre of computer game where a player explores dungeous, battles monsters, discovers treasure and undertakes quests. Larn is set around a single quest: to search for a potion to cure dianthroritis, a fatal illness that has stricken your daughter.

How do I get it?

My work is still in early days, so Larn is not yet ready for general play. I hope to one day offer simple binaries, and a web version.

If you have a working Go installation and Git installed on Linux or OS X (it needs ncurses), run 

$ go get github.com/dsymonds/larn
$ larn

or browse it on GitHub. If you're feeling particularly keen to try Larn, but can't build it yourself, send me some mail and tell me what OS you are running, and I'll try to send you a binary.

Send me mail if you have trouble. If you are feeling keen, and want to help with fixes or improvements, send me mail too.
